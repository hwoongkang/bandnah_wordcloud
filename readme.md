# 나씨밴 워드클라우드

## Dependencies

### venv 사용을 권장합니다.

```
$ python3 -m venv .venv
$ source .venv/bin/activate
(.venv) $ pip install -r requirements.txt
(.venv) $ python main.py
```

### jdk가 필요합니다.

https://www.oracle.com/java/technologies/downloads/

### 폰트 지정이 필요합니다.

main.py 12번째 줄을 otf나 ttf가 있는 위치로 변경해서 사용해 주세요!
